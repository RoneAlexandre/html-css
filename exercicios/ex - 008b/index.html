A tag <pre> servem para manter o texto pré-formatado, deixando-o vizualmente exatamente igual foi digitado, incluindo quebras de linhas, espaços e tabulações.

Citações Simples

A tag <q> serve para dar aspas ao texto indicando ser uma citação e consequentemente dar significado ao mesmo, tornando-o em samantico.

Citações Completas

A tag <blockquote> serve para criarmos uma citação mais longa. E vizualmente cria um espaço do texto para a borda. Podemos também colocar um link para indentificar de onde vem o texto original caso seja pego de algum outro lugar, isso usando a tag cite= dentro da tag blockquote.

Abrevições

A tag <abbr> serve para mostrar o significado da palavra abreviada, passando o mouse por cima dela podemos ver o significado escrito dentro da área title= da tag.

Texto Invertido

A tag <bdo> serve para criarmos um texto invertido, colocando rtr dentro da tag, o texto fica invertido e usando ltr o texto fica normal.
