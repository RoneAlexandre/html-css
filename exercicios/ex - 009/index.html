A tag <ol> serve para criar uma lista dentro de uma ordem.

Dentro desta tag temos o parametro type=, onde configuramos o tipo de marcador da lista atual.
Temos as seguintes opções: 1,A,a,I,i.
(números, letras maíusculas/minúsculas, números romanos maíusculos/minúsculos).

Também podemos indicar o início da contagem usando o parametro start dentro da mesma tag.

A tag <ul> serve para criar uma lista não ordenada.

Dentro desta tag também usamos o parametro type=, para configurar o estilo de marcador da lista, porém agora em formatos.
Temos as seguintes opções: disc, circle, square.

Lista de definições

Toda lista de definição esta dentro de uma tag <dl> (definition list).
Cada termo é um <dt> (definition term).
E cada descrição é um <dd> (definiton description).

HTML
Linguagem de marcação para a criação do conteúdo de um site.
CSS
Linguagem de marcação para a criação do design de um site.
JavaScript
Linguagem de programação para a criação de interatividade de um site.

Assim como os itens da lista, as tags <dt> e <dd> possuem fechamento opcional.
