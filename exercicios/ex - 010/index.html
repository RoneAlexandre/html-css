Usando links externos

Usamos a tag <a> para inserir um link dentro do site (âncora).

O principal atributo dentro desta tag é o href local onde inserimos o link desejado.
Dentro deste atributo colocamos uma URL completa para outro site.

Outro atributo que podemos usar para controlar onde o site vai abrir é o target que suporta os seguintes valores:

  • _blanck (vai abrir o link em uma nova janela em branco).
  • _self (vai abrir o link na janela ou frame atual (padrão).
  • _top (vai desfazer todos os frames e abrir o destino no navegador completo).
  • _parent (similar ao uso do _top em uma referência á janela mãe).
  • nome-do-frame (caso esteja usando frames, indicar o nome da janela abrir.)

Você pode acessar o meu perfil https://www.instagram.com/rone_alexandremt/ através deste link.

Tambem existe um recurso bem interessante para links que é indicar qual é a natureza do destino usando o atributo rel. Esse atributo aceita vários valores, entre eles temos:

  • next (indica que o link é para a próxima parte do documento atual).
  • prev (indica que o link é para a parte anterior documento atual).
  • author (indica que é um link para o site do autor do artigo atual).
  • external (indica que é um link para outro site que não faz parte do site).

Fazendo isso passamos a ter um local para clicar e ser enviado a próxima página, no caso de avançar para a próxima página, inserimos dentro do atributo rel o valor next.

Links para dowload

Para efetuar o dowload de algum material PDF ou de um arquivo ZIP qualquer, basta gerar o link diretamente do arquivo que se deseja efetuar o dowload e adicionar o atributo dowload com o valor configurado para o nome do arquivo a ser baixado e o atributo type para indicar ao navegador que tipo de arquivo está sendo baixado.
