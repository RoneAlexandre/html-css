Receitas(2)

Está é a minha página de receitas

Para retornar a página anterior utilizamos a mesma tag, porém agora dentro do atributo href basta inserir o nome do arquivo da primeira página. No caso de retornar uma página, inserimos dentro do atributo rel o valor prev.

→ Para voltar a página inicial basta clicar em voltar a página anterior e imediatamente retornará a aba anterior.

Para ir a terceira página que neste caso se encontra dentro de uma pasta, utilizamos a mesma tag, pórem agora devemos indicar dentro do atributo href o caminho (nome da pasta) antes do nome do arquivo.

→ Para ir para a página de noticias clique em próxima página e sera direcionado a próxima aba.
