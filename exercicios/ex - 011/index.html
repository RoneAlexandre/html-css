Imagens dinâmicas

Imagens diversas para ser utilizadas em diferentes dispositivos ou simplesmente para ser alteradas quando diminuimos ou aumentamos o nosso navegador.

Tags picture e source

A tag <picture> tem a função de concentrar outras fontes de imagem. Dentro dela colocamos a tag <img> para carregar a imagem que será usada como imagem padrão.
Acima da tag img, ainda dentro do aninhamento picture, inserimos uma outra tag chamada <source> (neste caso usaremos source:media:type) que possui três atributos:

  1. media: indica o tamanho máximo a ser considerado para carregar a imagem indicada no atributo srcset.
  2. srcset: vai indicar o nome da imagem que será carregada quando o tamanho indicado for atingido.
  3. type: vai indicar o media type da imagem que usamos.

É importante que exista uma ordem entre os source e nessa nossa configuração, os primeiros itens que se encontram mais acima sejam os de menores tamanhos determinado em max-width e que os seguintes sejam maiores, de uma forma crescente. O último item dentro de <picture> deve ser a imagem padrão desejada, inserida através da tag <img>

Reproduzindo Áudios

Vamos aprender a reproduzir áudios em HTML5

Tag <audio> e seus principais atributos.

Preload:
O atributo preload indica se o áudio será pré-carregado ou não, e aceita três valores;

  1. METADATA: vai carregar apenas as informações sobre o arquivo (tamanho, tempo, informações de direitos etc).
  2. NONE: não vai carregar absolutamente nada até que o usuário clique no botão play ou um script inicie a reprodução.
  3. AUTO: (padrão) vai carregar o arquivo de áudio inteiro assim que a página for carregada, mesmo que o usuário nunca aperte o play.

Controls:
O atributo controls vai apresentar o botão de player na tela. Caso não seja inserido dentro da tag <audio>, o controle será transparente e o usuário não poderá interagir com ele.

Autoplay
O atributo autoplay quando inserido, vai iniciar a reprodução automaticamente do áudio assim que a página for carregada.

Loop
O atributo loop quando inserido, tem a função de fazer com que o áudio seja repetido eternamente, assim que terminar a sua reprodução ele iniciará automaticamente de novo.

Source dentro da tag < audio>.

Dentro da tag audio e seus atributos, podemos adicionar vários <source> indicando formatos diferentes do mesmo áudio. Colocamos primeiro o formato de áudio favorito. Os demais inseridos em seguida, só serão carregados caso o primeiro falhe.
Caso todos falhem, criamos um parágrafo disponibilizando o download do arquivo MP3 para que o usuário baixe e ouça no player padrão de seu dispositivo.

No caso da utilização de áudio usamos source:src type onde dentro de src inserimos o arquivo desejado e em type o estilo de áudio que estamos utilizando.
