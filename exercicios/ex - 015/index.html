Externo

No estilo externo utilizamos a tag <link> (neste caso usamos link:css) dentro da área <head>do nosso documento HTML.

Tag <link>

A tag link carrega a função de criar uma nova aba para ser especialmente utilizada para trabalhar com as configurações de estilos CSS. Sendo assim não precisaremos mais fazer a configuração de página por página.
